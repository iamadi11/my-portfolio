# ship

Run this when you want a **single high-impact portfolio fix**.

## Steps

1. Read `SOURCE_OF_TRUTH.md` and `CURSOR_RULES.md`.
2. **Audit** quickly:
    - Hero: WHO + WHY in ~5s? Credible? Motion where it helps?
    - Projects: GitHub-backed? Remove or demote weak entries.
    - Experience / contact: matches resume?
    - Skills: provable?
3. **Pick one issue** with the highest ROI (credibility → first impression → conversion → UX → performance).
4. **Implement only that fix** (one logical change).
5. Run `npm run build`; sanity-check mobile + desktop.
6. Commit: `feat(portfolio): … — based on resume` (or LinkedIn/GitHub as appropriate).

## Stop condition

Stop after **one** shipped fix unless the user asks for more in the same turn.
